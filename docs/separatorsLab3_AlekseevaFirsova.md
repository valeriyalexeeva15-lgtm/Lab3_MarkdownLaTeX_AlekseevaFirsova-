Markdown - это простой язык разметки для быстрого форматирования текста (заголовки, списки, жирный/курсив), который легко преобразуется в НТМL и используется для создания документации, постов в блогах и веб-контента,
позволяя сосредоточиться на тексте, а не на сложном коде
___
Основные возможности Markdown это быстрое и простое форматирование текста (заголовки, списки, ссылки, выделение), сохраняющее читабельность, для последующего преобразования в НТМL или другие форматы, идеальное для документации, заметок, README-файлов, блогов и сообщений, с синтаксисом, использующим обычные символы вместо сложного кода, что делает его удобным и интуитивно понятным
___
Пример форматирования. Заголовки:
# Заголовок 1
## Заголовок 2
### Заголовок 3
___
